Insanity library architecture

 

 

 

 

 

 

 




 

 

 

The lines            specify the way and direction, how each module controls other module(s). They lead from higher-level modules towards lower-level modules.
This means that particular module is dependent on all modules, which it controls, directly or indirectly using other modules.

Dotted lines         represent connections with many other modules (if not all), so practically the modules towards which multiple dotted lines lead are essential for proper work of the rest of the system.
