S_vector

S_vector is a 3-element vector used for storing 3D location, normals, directions, RGB color values or for other purposes.
The class is declared as:

struct S_vector{
   union{
      struct{
         float x, y, z;
      };
      float f[3];
   };
};

float GetBrightness() const;

 

Get brightness of vector, assuming vector represents an RGB value.

 

Return value:

 

The returned value is the brightness, using the formula:

brightness = RED * 0.3 + GREEN * 0.6 + BLUE * 0.1,

where x = RED, y = GREEN and z = BLUE.
