S_float_random

This class is used as a random number generator of floating values.
Two constructors are used for generation of random values.
A conversion operator is used to convert value from this class to float number.

This class is is declared as follows:

struct S_float_random{
   float f;
   S_float_random();
   S_float_random(float max);
   operator float() const;
};

 
 

S_float_random();

This is a constructor used for generation of random floating-number value from 0.0f to 0.99999f.

S_float_random(
                  float max
);

This is a constructor used for generation of random floating-number value from 0.0f to specified maximum.

Parameters:

max

Maximal number, up to which random number may be generated.

operator float() const;

Operator used for retrieving the data from the class.
