S_int_random

This class is used as a random number generator of integer values.
One constructors are used for generation of random value.
A conversion operator is used to convert value from this class to integer number.

This class is is declared as follows:

struct S_int_random{
   int i;
   S_int_random(int max);
   operator int() const;
};

 
 

S_int_random(
               int max
);

This is a constructor used for generation of random integer number value from 0 to specified maximum minus one.

Parameters:

max

Maximal number, up to which random number may be generated, non-inclusive.

operator int() const;

Operator used for retrieving the data from the class.
