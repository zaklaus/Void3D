S_quat

S_quat Slerp(

            const S_quat &q,

            float t,

            bool shorten = false

) const;

 

Interpolate between two quaternions.

 

Parameters:

 

q

 

Destination of interpolation.

 

t

 

Interpolation ratio, in range from 0.0 to 1.0, where value 0.0 returns source quaternion (this), and 1.0 returns destination quaternion q. Any value between these extremes returns interpolated quaternion.

 

shorten

 

Flag specifying if shorter path should be selected, if angle among quaternions is more than 180 degrees.

 

Return value:

 

Resulting interpolated quaternion.

void SlerpFast(
            const S_quat &q,
            float t,
            S_quat &result
) const;

 

Interpolate between two quaternions  - version optimized for speed. The results are similar like with Slerp, but the interpolation is only approximated, so the results may differ.

 

Parameters:

 

q

 

Destination of interpolation.

 

t

 

Interpolation ratio, in range from 0.0 to 1.0, where value 0.0 returns source quaternion (this), and 1.0 returns destination quaternion q. Any value between these extremes returns interpolated quaternion.

 

result

 

Quaterion, where the result of operation will be stored to.

void I3DAPI Scale(
            float s

);

 

Scale quaternion by factor. This is the same as interpolating using Slerp, where destination quaternion is identity.
 
Parameters:
 

s

 

Interpolation ratio, in range from 0.0 to 1.0, where value 0.0 returns source quaternion (this), and 1.0 returns identity quaternion. Any value between these extremes returns interpolated quaternion.
