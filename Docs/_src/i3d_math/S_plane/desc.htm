S_plane

Plane is represented in parametric form - plane's normal and d part of equation:

P . N + d = 0

P is any point on plane, N is plane's normal.
P.N is dot-product of those two vectors.

The structure is declared as follows:

struct S_plane{
   S_vector normal;
   float d;
};

bool IntersectionPosition(
               const S_vector &pos,

               const S_vector &dir,

               float &ret

) const;

 

This function computes intersection of this plane with line.
 

Parameters:

 

pos, dir

 

Line, for which intersection will be computed.

 

ret

 

Variable, which will contain relative position on line where collision occurred, if the call succeeds.
To get intersection point, you can use this value in this equation:

S_vector xpt = pos + dir * ret

 

Returned value:

 

The returned value is true if intersection was successfully computed.

  

  
  
If plane and line are parallel, the returned value is false.
