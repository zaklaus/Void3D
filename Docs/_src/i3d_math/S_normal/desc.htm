S_normal

S_normal inherits from  S_vector class, and it is used for unit-length vectors.
S_normal is defined as follows:

struct S_normal: public S_vector{

};

Constructors:

S_normal();

Description:

S_normal();

 

Default constuctor;

S_normal(

            const S_vector &u,

            const S_vector &v

);

 

Construct unit-length normal from 2 direction vectors (using cross-product).

 

Parameters:

 

u, v

 

Vectors, from which normal vector will be computed using cross-product.

 

Remarks:

 

The magnitude of resulting normal vector will be one.

S_normal(

            const S_vector &v

);

 

This constructor is used for normalization of input vector.

Parameters:

 

v

 

Vector, from which normal vector will be computed.
