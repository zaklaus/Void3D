S_vector2

S_vector2 is a 2D counterpart of S_vector. Most functions of S_vector are also implemented for S_vector2, and their meaning is the same, but limited to 2 dimensions.

Parameters:

Second line.

 

Return value:

If lines are parallel or coincident, the returned value is false.

float GetBrightness() const;

 

Get brightness value from RGB. This function is intended for cases when S_vector holds RGB information.

 

Return value:

 

Brightness computed from RGB information by following formula:

  brightness = x * .3f + y * .6f + z * .1f
