INET_INIT

INET_INIT structure is used when creating the I_net interface. The structure is defined as follows:

struct INET_INIT{
   dword flags;
   const void *app_guid;
   const char *session_name, *player_name;
   IN_CONNECTION connection;
   dword max_players;
   const char *modem_phone_number;
   const char *ip_host_name;
   const char *port;
   int drop_timeout; 
   typedef void __stdcall t_log_func(const char *msg);
   t_log_func *log_func;
 
   enum E_VOICE_COMPRESSION{
      VS_NULL,
      VS_1_2,
      VS_3_2,
      VS_6_4,
      VS_8,
      VS_13,
      VS_32,
      VS_64,
   } voice_compression;
};

typedef INET_INIT *PINET_INIT;

Members:

flags

Initialization flags. Following flags are defined:

ININIT_CLIENTSERVER

Use client-server model, rather than peer-to-peer model. This mode is currently not implemented.

ININIT_MIGRATE_HOST

Allow migration of session host. If this flag is not specified, and host machine terminates session, the session is lost.

app_guid

Unique identifier of application - buffer containing 16 bytes.

session_name

Name of the session, if session will be hosted by this interface.

player_name

Name of local player, when connection will be established.

connection

ID of connection provider. Available providers are following:

INCON_TCPIP

TCP/IP connection

INCON_IPX

IPX connection

INCON_MODEM

modem connection

INCON_SERIAL

serial cable connection

INCON_SIM

simulated TCP-IP connection

Currently, only TCP/IP connection is implemented.

max_players

Maximal number of players allowed to join the session. Specify 0 for unlimited amount.
This parameter is used only if session is hosted.

modem_phone_number

Pointer to string specifying telephone number, may be NULL if hosting session (waiting to be dialed).

ip_host_name

Pointer to string specifying IP address of hosting machine, may be NULL for LAN.

port

Pointer to string specifying port number, may be NULL if default port is being used.

drop_timeout

Time-out value, after which we discover a player is dropped

log_func

Pointer to logging callback function, which will be called whenever a log is being sent.

voice_compression

Enumerated type, specifying bit-rate of voice chat session. The voice chat session is initialized later by special methods of I_net interface.
