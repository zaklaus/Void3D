C_net_out_msg

C_net_out_msg is an interface for constructing outgoing (compacted) message. The message consists of guaranteed and non-guaranteed sub-messages, which are sent independently, using different network options.
Messages are constructed from bit parts of various bit depth, from 1 to 32 bits per element. This is due to saving bandwidth.
User is also responsible for creating contents of outgoing message and sending it out in appropriate time.

void Ascend();

Ascend - create new chunk. This creates chunks in both guaranteed and non-guaranteed sub-messages. Parameter id specifies identification of chunk, which is always a word value.
Chunks may be nested.

Remarks:

If chunk with the same ID is already on tail of the message, the chunks are concatenated into one single chunk, in order to save bytes associated with chunk management. Each chunk consumes 4 bytes of message (2 bytes for chunk ID and 2 bytes for chunk size).

void Descend();

 

Descend - close chunk. This determines chunk size and writes it into chunk header. If chunk size is zero, chunk is discarded; so empty chunks are not sent over network. This way user may open and close chunks without knowing if any information will be written into it.

void Clear();

 

Clear the contents of message.
