E_NET_STATUS

This enumerated type represents the status of I_net interface.
Following values are declared:

INET_ST_UNINITIALIZED

The interface is uninitialized.

INET_ST_UNCONNECTED

The interface is not connected - connection either not established, lost or closed.

INET_ST_CONNECTING

Currently connecting - when connecting is finished, the state automatically (asynchronously) switches to INET_ST_CONNECTED or INET_ST_UNCONNECTED state.
You may use Close method to stop connection attempt.

INET_ST_ENUMERATING_SESSIONS

Currently enumerating sessions - in this state it is allowed to use GetSessions method.
You may use Close to stop the enumeration, or JoinSession to try to connect.

INET_ST_CONNECTED

Connected - in this state it is possible to send messages, or use Close to disconnect.
