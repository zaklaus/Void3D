INET_MESSAGECALLBACK

This is a definition of callback function, which is used to retrieve input messages sent from other players. It is up to user to declare this function and perform appropriate actions on received data.

bool INETAPI INET_MESSAGECALLBACK(

            I_net *inet,

            C_net_in_msg *in_msg,

            dword context

);

Parametrs:

inet

Pointer to network driver interface, which processes the message.

in_msg

Data being received, stored in C_net_in_msg interface.

context

User-defined parameter passed in call to ProcessMessages.

Return value:

If the user returns true, further pending messages are also retrieved from the queue, until the input queue is empty. If user returns false, the message processing is stopped after the call returns to ProcessMessages. In that case, user may retrieve other messages in the input queue by calling ProcessMessages again.

See also:

I_net, C_net_in_msg, I_net::ProcessMessages
