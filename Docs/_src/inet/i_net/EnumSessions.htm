I_net::EnumSessions

Get list of enumerated sessions.

bool EnumSessions(

            PINET_session_address *addrs,

            dword *num_sessions

);

Parameters:

addrs

Buffer to fill addresses to, parameter num_sessions is the size of this buffer. If the buffer is not big enough, the call fails, and variable pointed to by num_sessions contains number of available addresses.

num_sessions

Pointer to a variable specifying the size of buffer pointed to by addrs parameter.

Return value:

The return value is true if enumeration was successful and all available addresses were copied into the buffer. If buffer is not nig enough, the call fails.
If the call succeeds, the parameter num_sessions contains the number of items actually copied to the buffer.

Remarks:

Because the network interface works in multithreaded mode, all addresses copied to the buffer have increased one reference counter. After retrieving addresses, user must call Release on each of addresses.
It is convenient to retrieve the addresses into an array of smart pointers, which automatically release the reference, when the buffer is destroyed.

Note that the number of available sessions may change between subsequent calls to EnumSessions, so user should call this function in a loop until not successful.
