C_net_in_msg

C_net_in_msg - interface for receiving and dispatching incoming message.
This interface is created by system when new message is received, and is passed to user callback function for processing.

Descriptions:

byte GetByte();

 

Get byte value from message.

word GetWord();

 

Get word value from message.

dword GetDword();

 

Get dword value from message.

float GetFloat();

 

Get float value from message.

class S_vector GetVector();

 

Get vector value from message.

word GetID();

 

Get identification value from message.

dword GetBits(

            dword num_bits

);

 

Get specified number of bits from message. The bits are returned starting in lowest possible bit of returned 32-bit value.

void GetString(

            char *buf,

            dword max_len

);

 

Get null-terminated string from message. Entire string is always read from message, regardless of the size of input buffer, but not more than max_len characters are copied into the buffer. The buffer is always terminated by null character.

void Ascend();

 

Ascend - step into chunk. The step is made after chunk's identification value is being read.

void Descend();

 

Descend - step out of chunk. The message read pointer is moved after the chunk.

void SkipChunk();

 

Skip chunk. The skip is made after chunk's identification value is being read.

void Skip(

            dword num_bits

);

 

Skip specified number of bits.

bool ChunkEnd() const;

 

Determine if read pointer is at the end of currently opened chunk.

dword GetPos() const;

 

Get current read position, in bits.

void *GetBuf() const;

 

Get direct pointer to the message buffer.

dword GetSenderPID() const;

 

Get ID of player who has sent this message. This is a valid ID of network player, or 0 if the message in the buffer is a system message. See INET_SYSTEM_MESSAGE for details.

dword GetSize() const;

 

Get size of the message, in bits.
