InetCreate

This function creates the I_net interface and initializes it.

I_net *INETAPI InetCreate(

            INET_INIT *init_data,
            bool use_debug_version
);

 

Parameters:

 

init_data

 

Pointer to INET_INIT structure, which contains initialization data required for initialization of the interface.

 

use_debug_version

 

This parameter specifies if Debug or Release dynamic-link library should be loaded. If passing true, a debug version is loaded, otherwise a release version is loaded.

           

Return value:

 

If the call succeeds, the returned value is pointer to initialized I_net interface.
If the call fails, the return value is NULL.
