table_callback

The table_callback type is definition of a callback function, which is used during editing of table. It is declared as:

typedef void (TABAPI *table_callback)(
            PC_table tab,
            dword msg,
            dword cb_context,
            dword prm2,
            dword prm3
);

Parameters:

tab

Pointer to C_table interface, which is being edited.

msg

Message being posted to the callback function. This may be one of following:

TCM_MODIFY

Sent when value in table has been modified. Parameters are: prm2 = dword item_index; prm3 = dword array_index.

TCM_IMPORT

Entire table was imported from file.

TCM_CLOSE

Dialog is about to be closed. Parameters are: prm2 = (RECT*)pos_size (last position and size of dialog window).

TCM_GET_ENUM_STRING

A query is made to application for providing names of enumerated type. Parameters are: prm2 = dword item_index; prm3 = (C_str*)ret.
This message is sent only when contents of enumerated type is displayed, for which template has no names of enumerated types. The application’s responsibility is to fill-in null-terminated strings, terminated by additional null character, into provided ret parameter (passed in prm3 parameter of the callback function.
Note: to assign multiple null-terminated strings into C_str class, use C_str::Assign method.

cb_context

User-defined context value, which was passed to C_table::Edit method.

prm2, prm3

Value, whose usage is dependent on message being sent. See description of msg member for details.

C_table, C_str
