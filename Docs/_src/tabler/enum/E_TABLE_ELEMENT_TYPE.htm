E_TABLE_ELEMENT_TYPE

This enumerated type is used for definition of table e

lement types, and also contains additional types used for construction of table template. The enumerated type is defined as follows:

enum E_TABLE_ELEMENT_TYPE{
   TE_NULL,                   //must be last member of template

   TE_BOOL,
   TE_INT,
   TE_FLOAT,
   TE_ENUM,
   TE_STRING,
   TE_COLOR_RGB,
   TE_COLOR_VECTOR,

   TE_BRANCH,
   TE_ARRAY
};

 

Type description:

TE_NULL is used for marking an invalid type (returned by some functions of C_table interface. It is also used in table template to mark the end of array of table elements.

TE_BOOL to TE_COLOR_VECTOR are actual data types contained in the table.

TE_BRANCH is special type, used only during editing. It encloses logical block of other elements into one expandable/collapsible tab in the editor dialog.

TE_ARRAY is used before other type to mark that following type is an array of elements, not a single element.
