C_table_template

C_table_template class is used for definition of table template, which is used for two purposes: initialization of C_table data layout using its Load method, and for editing the data stored in the C_table using the Edit method.
The class is defined as:

class C_table_template{
public:
   const char *caption;
   const C_table_element *te;
};

typedef C_table_template *PC_table_template;
typedef const C_table_template *CPC_table_template;

Members:

caption

Caption of table – used during editing for name of dialog window.

te

Pointer to array of table element classes (see C_table_element for details), which define the layout of the table. The array must be terminated by element, which has first member set to TE_NULL type, marking the end of the template array.

C_table, C_table_element
