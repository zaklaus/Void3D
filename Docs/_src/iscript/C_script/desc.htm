C_script

This interface is used for compilation of script in text form, and keeping it in memory. The script may be loaded into virtual machine C_v_machine.

C_script::Compile

Compile script from source text file.

ISL_RESULT Compile(

            const char *fname,

            dword flags = 0,

            const char * cmd_line = NULL,

            T_report_err *cb_err = NULL,

            void *cb_context = NULL

);

Parameters:

fname

File name of script, which is to be compiled.

flags

Compilation flags. Following flags are defined:

ISLCOMP_VERBOSE

Dump debug information during compilation.

ISLCOMP_PRECOMPILE

Save precompiled file, or use if available and dates match (see Remarks for details).

ISLCOMP_FORCE_PRECOMPILED

Force use of precompiled files (see Remarks for details).

ISLCOMP_PRECOMPILE_HIDDEN

Set hidden file flag on generated .pcp precompiled files.

cmd_line

String specifying command-line switches. Following switches are defined:

/i<path>   add path into include-files search paths

/I<filename>   force processing of include file (the include files on command-line are parsed before the script is compiled, and in the order in which they appear in the line)

cb_err

Pointer to callback function, which will receive error and warning messages encountered during compilation. The function is defined as:

void ISLAPI ErrReport(
  const char *msg,
  void *context,
  int line,
  int col,
  bool warn
);

Parameters:
msg … error message
context … context value passed into Compile method
line, col … line and column indices specifying position, where the error occurred
warn … true if the message is warning, false if error

cb_context

User-defined context value, which is passed into the error-reporting callback function.

Return value:

The returned value is I3D_OK if the call was successful, or other error return code marking an error.
Additionally, the returned value may be ISL_UPTODATE if script was loaded from precompiled .pcp file.

Remarks:

Precompiled scripts are used, when user specifies ISLCOMP_PRECOMPILE or ISLCOMP_FORCE_PRECOMPILED flags. The files containing precompiled scripts are stored on disk with original name, but with .pcp extension instead of file extension passed to the Compile method. They may be created first time when script is compiled, and loaded from dist next times when the same script is compiled. This saves compilation times, and also allows delivery of compiled scripts to target audience.
The Compile method automatically checks the date-time of script file and all include files it references. If any of them is newer than date-time stored in the .pcp file, the script is compiled from text source, and .pcp file is recreated.
When ISLCOMP_FORCE_PRECOMPILED flag is specified, only precompiled version (with .pcp extension) is loaded. This is suitable for shipping version of application, where only precompiled script versions may be shipped.

C_script::GetStatus

Get current status of the interface

ISL_RESULT GetStatus () const;

Return value:

The compilation status of the script. The returned value is ISL_OK if interface contains compiled script program, or ISLERR_NOPRG otherwise.

C_script::GetAddress

Return address of script’s symbol in global scope. The symbol may be either variable name, or function name.

const void *GetAddress(

            const char *sym_name

) const;

Parameters:

sym_name

Name of symbol, whose value is queried.

Return value:

The returned value is const pointer to memory, where function/variable begins. If the symbol was not found, the returned value is NULL.
Note that it is not allowed to write into provided pointer.

C_script::GetTableTemplate

Return pointer to compiled table template.

const C_table_template *GetTableTemplate() const;

Return value:

The returned value is const pointer to template representing the table stored inside the compiled script. If script has no table, the returned value is NULL.

C_script::Dump

Dump listing of assembled code (in pseudo-instructions). The dumping is done by calling provided callback function with each dumped text line.

bool Dump(

            void(ISLAPI *func)(const char*, void *context),

            void *context

) const;

Parameters:

func

Pointer to callback function, which will be called with each line of dumped code.

context

Context value passed to the callback function.

Return value:

The returned value is true if code was dumped correctly, or false if some error occurred (e.g. invalid opcode, or other problem in compiled script).
