IScript library

The scripting library is a powerful tool for implementing scripts performing particular tasks. The scripting language is a C-like language, which is precompiled using script compiler, and run in a virtual processor.

Library: IScript.dll, IScriptd.dll
Header file: IScript.h

Interfaces:

C_script

C_v_machine

The system is based on two main interfaces: C_script and C_v_machine.
C_script is used for compilation of source scripts (in text form), and keeping them in memory. After script is compiled, the class acts as read-only interface, which may be a source into one or multiple virtual processors.
C_v_machine is a class used for execution of compiled script in its own environment. Each virtual machine holds is capable of running one compiled script at a time. It holds all information about current state of the script (script-global variables, stack, table data, etc).
