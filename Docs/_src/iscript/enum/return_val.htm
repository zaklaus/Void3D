IScript return codes

Following are return codes from IScript library functions:

Success codes:

ISL_OK

This code is returned when the method call was successful.

ISL_UPTODATE

Compiled precompiled file (.pcp) was up-to-date, and was loaded instead of compiling script.

ISL_SUSPENDED

Program run in virtual machine was suspended.

Compiler errors:

ISLERR_NOFILE

File was not found.

ISLERR_BADSCRIPT

An error encountered durnig compilation.

ISLERR_GENERIC

Internal error.

ISLERR_INVALIDOPTIONS

Command-line options are invalid.

Load errors:

ISLERR_NOEXTSYM

External symbol cannot be located during linking program.

ISLERR_NOPRG

No program is currently loaded.

Program run errors:

ISLERR_INVOPCODE

Invalid op-code encountered during program run.

ISLERR_STACKCORRUPT

Stack corrupted during program run.

ISLERR_INVADDRESS

Invalid address specified.

ISLERR_INVPARAM

Invalid input parameter(s).

ISLERR_NOT_FOUND

Object not found.

ISLERR_BAD_TABLE_INDEX

Invalid index into table.

ISLERR_DIVIDE_BY_ZERO

Dividing by zero encountered.

ISLERR_NEVERENDING_LOOP

Neverending loop detected (too many continuous instructions executed).
