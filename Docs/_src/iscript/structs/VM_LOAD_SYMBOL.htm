VM_LOAD_SYMBOL

VM_LOAD_SYMBOL structure is used for loading of script into virtual machine. It contains information about single link symbol. Multiple structures in array are used for definition of multiple link symbols for loaded program.

struct VM_LOAD_SYMBOL{
   void *address;
   const char *name;
};

Members:

address

Address of symbol. NULL value is used for specifying the end of array of symbols.

name

Name of symbol.
If this value is NULL, and address member is not NULL, the address is used as pointer into another array of VM_LOAD_SYMBOL structures. This way user may include link symbols from multiple sources.
