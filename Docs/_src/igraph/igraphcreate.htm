IGraphCreate

This function is used for creation of IGraph interface and its initialization.

LPIGraph I2DAPI IGraphCreate(

            LPIG_INIT is,

            bool use_debug_version

);

 

Parameters:

 

is

 

Pointer to IG_INIT structure, which contains initialization data required for initialization of the interface.

 

use_debug_version

 

This parameter specifies if Debug or Release dynamic-link library should be loaded. If passing true, a debug version is loaded, otherwise a release version is loaded.

           

Return value:

 

If the call succeeds, the returned value is a pointer to newly created IGraph interface.
If the call fails, the return value is NULL.
