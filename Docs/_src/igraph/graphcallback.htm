IGraph Callback function

 

The callback function is used when registering callback to IGraph interface. This type is a definition of callback function.

Registered callback functions are called in certain system circumstances – see description of msg parameter to learn which messages may be sent to callback functions.

 

typedef dword GRAPHCALLBACK(

            dword msg,

            dword par1,

            dword par2,

            void *context

);

Parameters:

msg

Message sent to callback function. This may be one of following:

 

CM_ACTIVATE

 

Message sent when application widow is activated or deactivated. Parameter prm1 is a boolean status, representing if application was activated (true) or deactivated (false).

 

CM_COMMAND

 

User has selected a command from Windows menu. Parameter prm1 specifies ID of selected menu item.

 

CM_RECREATE

 

Window is being re-created as result of changing IGraph parameters (IGraph::UpdateParams), re-sizing the window, or reset on DirectX device.

Meaning of parameters:

prm1

New IGraph flags. See IGraph initialization flags for details.

prm2

Phase of re-creation. This may be one of following:

0 … Message sent before destroying DirectX devices. All hardware resources must be released.

1 … Message sent after creating new DirectX devices.

2 … Message sent to notify registered functions about applying changes.

 

CM_PAINT

 

A Windows message WM_PAINT is being processed, so by this message IGraph informs registered callbacks that window contents should be painted.
This message is typically sent when main window is suspended, and area of application window must be updated in such suspended state.

par1, par2

Parameters to callback function. For meaning of these parameters, see msg parameter.

context

Context value which was specified in call to IGraph::AddCallback.

Return value:

The returned value is true, if function was registered successfully.
If the function already is registered by the IGraph interface, the returned value is false.

See also:

 

IGraph::AddCallback, IGraph::UpdateParams, IGraph initialization flags
