IG_INIT

This structure is used in call to IGraphCreate function for specifying parameters of the graphics window.
The structure us defined as follows:

struct IG_INIT{
   dword pos_x;
   dword pos_y;
   dword size_x;
   dword size_y;
   dword bits_per_pixel;
   dword flags;
   int adapter_id;
   typedef void __stdcall t_log_func(const char*);
   t_log_func *log_func;
   enum E_ANTIALIAS_MODE{
      ANTIALIAS_NONE,
      ANTIALIAS_NORMAL,
      ANTIALIAS_BEST,
   } antialias_mode;
};

typedef IG_INIT *LPIG_INIT;

Members:

pos_x, pos_y

Required position of client area of the main window. These members are used only if IG_INITPOSITION flag is also specified.

size_x, size_y

Required resolution of client area of the main window. In fullscreen modes, this must match to a resolution of available video modes.

bits_per_pixel

Requested bit-depth of primary surface in bits per pixel. This value is ignored for windowed modes.

flags

See IGraph Initialization Flags for details.

adapter_id

DirectX adapter identifier.

log_func

Pointer to a log functions. If this is NULL, log output is disabled. This user-defined function may log information it receives whatever way it needs to.
This log function is used in cases when user request debug information.

antialias_mode

A member of  enumerated type, specifying which antialiasing technique, if any, will be used during rendering into the window.
