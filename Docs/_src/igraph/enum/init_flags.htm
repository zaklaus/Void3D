IGraph initialization flags

Initialization flags are used during initialization of IGraph interface, stored in the IG_INIT structure.
Following flags are currently defined:

IG_FULLSCREEN

The screen will be initialized in full-screen mode.

IG_TRIPPLEBUF

Triple buffering is requested. This request may silently fail, if the system doesn’t have enough system memory for additional swap buffer. In such case, a single back-buffer is used.

IG_NO_VSHADER

Disable creation of device supporting hardware vertex shaders (create software-vertex-processing device).

IG_NOSYSTEMMENU

Don't create system menu on window.

IG_NOMINIMIZE

Disable minimize button on window.

IG_RSIZEABLE

The window will be resize-able.

IG_SUSPENDINACTIVE

Pause application when it loses focus. This option sleeps current application whenever it is not the active window, and lets other application to use 100% of CPU time during these periods.

IG_NOTITLEBAR

Do not create title bar on window.

IG_NOHIDEMOUSE

Don't hide Windows cursor in exclusive mouse mode. Normally, when mouse is initialized by application, the cursor is hidden and it is up to application to display its own cursor. If this flag is set, the mouse cursor is never hidden.

IG_DEBUGMOUSE

Mouse mode more suitable for debugging. When mouse is initialized in this mode, it is not grabbed exclusively by application, rather system cursor is hidden and mouse works like in normal mode. When application is debugged in a debugger, mouse cursor is not captured, and may be normally used.

IG_LOADMENU

Load menu. If user wants to use Windows menu in application, one should specify IG_LOADMENU flag in call to IGraphCreate. Menu must also be present in application's resources with name "MENU_MAIN".

IG_HIDDEN

When creating window, create it hidden. Application is responsible for un-hiding the window when appropriate.

IG_VERBOSE

Dump debug information about underlying DirectX device to log ostream.

IG_WARN_OUT

When this interface is released, and this initialization flag is specified, it check if all interfaces created by its methods are released too, and dumps warning message about number of unreleased interfaces. This flag is intented for debug purposes only.

IG_INITPOSITION

Initialize position of window by members pos_x and pos_y of initialization structure. When this flags is not specified, the window is placed at the same position as it had when this application was running.

IG_REF_DEVICE

Select reference Direct3D device.
