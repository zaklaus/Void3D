IGraph::UpdateScreen

This method is used to update the contents of backbuffer to frontbuffer. Depending on how IGraph interface was initialized, it may do so by copying entire content of backbuffer to frontbuffer (windowed mode), or by swapping the frontbuffer with backbuffer.
This method always updates entire contents of backbuffer, and is intended to be used after you’re finished with rendering graphics into the backbuffer.

bool UpdateScreen(

            dword flags = 0,

            void *dest_hwnd = NULL

);

 

Parameters:

 

flags

This parameter is currently not used and must be zero.

dest_hwnd

Handle of destination window (HWND).
If this parameter is NULL, standard swapping mechanism is used as described above.
If this parameter is not NULL, the content of backbuffer is copied into client area of specified window.

Return value:

The method returns true if the call succeeds, or false if the update failed.
