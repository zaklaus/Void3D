IGraph:: UpdateParams

This method is used for changing parameters of IGraph interface. This function may change resolution of window, pixel-depth or update creation-flags.

bool UpdateParams(

            dword sx,

            dword sy,

            dword bits_per_pixel,

            dword flags,

            dword flg_mask,

            bool force_recreate = false

);

 

Parameters:

sx, sy

New resolution of windows or fullscreen mode. If both values are set to zero, screen resolution is not modified. After changing screen resolution, the viewport is reset to full screen area.

bits_per_pixel

New pixel-depth, in bits per pixel. This value is applied only in full-screen mode. If this value is zero, the pixel-depth is not modified.

 

flags

 

New combination of IGraph initialization flags. This parameter is used with flg_mask.

 

flg_mask

 

Mask of flags, which have to be modified, and for which flags parameter defines new values.

 

force_recreate

 

Set this value to true to force re-initialization of interface although screen resolution.

           

Return value:

 

The method returns true if update was completed successfully.
