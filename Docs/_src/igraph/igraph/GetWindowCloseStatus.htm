IGraph::GetWindowCloseStatus

 

This method returns the status of window’s Close button. Whenever this button is clicked, or Alt+F4 keys are pressed, the internal boolean variable is set to true. Calling this method use may read this variable, and clear the state of the variable. Appropriate action for close request is responsibility of application.

 

char GetWindowCloseStatus() const;

 

Returtn Value:

 

If user clicked Close button or pressed Alt+F4 keys since last call to this function, the returned value is true.
