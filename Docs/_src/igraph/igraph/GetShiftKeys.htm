IGraph::GetShiftKeys

This method returns status of modifier keys. These keys are Shift, Ctrl and Alt.

dword GetShiftKeys() const;

 

Return Value:

Bits representing if modifier keys are pressed. Any combination of the modifiers may be pressed.
The symbolic names for bits representing the keys are as following:

SKEY_SHIFT ... Shift key
SKEY_CTRL ... Ctrl key
SKEY_ALT ... Alt key
