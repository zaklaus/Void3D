IGraph::GetKeyName

This method is used for retrieving user-friendly name of keyboard key.

dword GetKeyName(

            IG_KEY key,

            char *buf,

            dword buf_size

);

 

Parameters:

key

Key whose name is to be retrieved.

buf

Pointer to buffer, where name of key will be copied.

buf_size

Size of buffer pointed to by buf. The number of characters copied will be less or equal to buf_size.

Return value:

The returned value is number of characters copied to the buffer, including the terminating null character.
