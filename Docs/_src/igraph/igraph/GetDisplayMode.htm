IGraph::GetDisplayMode

This method retrieves one of available display modes, supported by graphics driver, which are suitable for use by IGraph interface.

bool GetDisplayMode(

            int  id,

            dword *sx,

            dword *sy,

            dword *bpp

) const;

 

Parameters:

 

id      

Parameter specifying index of mode to get information about. This value must be in range from zero to number of available display modes minus one. To obtain number of available display modes, use NumDisplayModes method.

sx, sy

Pointers to variables, which will contain width and height of display mode, in pixels, when the call returns successfully.

bpp

Pointers to variable, which will contain bit-depth of display mode, in bytes, when the call returns successfully.

Return value:

The returned value is true if call was successful.
If particular display mode couldn’t be retrieved (i.e. invalid index was specified), the returned value is false.
