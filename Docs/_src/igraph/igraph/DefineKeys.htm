IGraph::DefineKeys

 

This method sets up table of game keys - keys which may be pressed simultaneously. The status of all defined key is determined in one call to ReadKeys method.

 

void DefineKeys(

            const IG_KEY *keys,

            dword num = 32

);

 

Parameters:

 

keys

 

Pointer to array of key codes. Each key is represented by a symbolic constant. See Keyboard Constants for list of keyboard codes. If some position doesn't need to be tested, set the value to K_NOKEY.

 

num

Number of values in array.

Remarks:

 

Each key in the array has assigned a bit in 32-bit dword value returned by ReadKeys method. Key on offset 0 has assigned bit #0, and so on. Bit with value '1' means that key is pressed, the value '0' means non-pressed keys.
