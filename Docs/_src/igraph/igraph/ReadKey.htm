IGraph::ReadKey

 

Return recently pressed key.

 

IG_KEY ReadKey(

            bool process_messages = false

);

 

Parameters:

 

process_messages

If this flag is true, the method internally calls ProcessWinMessages.

Returtn Value:

 

A key code. See Keyboard Constants for list of keyboard codes

 

Remarks:

 

When this method returns, the key is removed from buffer, so subsequent calls will return K_NOKEY, if no new key was pressed.

 

See also:

 

ProcessWinMessages, Keyboard Constants
