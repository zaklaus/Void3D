IGraph::AddCallback

 

This method adds function to call-back list. The IGraph interface communicates with program using it by calling registered callback functions when critical condition is encountered, for example when display mode changes, display device is reset, etc.

 

bool AddCallback(

            LPGRAPHCALLBACK cb_proc,

            void *context

);

Parameters:

      cb_proc

Pointer to a callback functions, which is of type GRAPHCALLBACK.

      context

User-defined value which will be passed to the callback function, when it is called.

Return value:

The returned value is true, if function was registered successfully.
If the function already is registered with the same context value, the call fails and returned value is false.
