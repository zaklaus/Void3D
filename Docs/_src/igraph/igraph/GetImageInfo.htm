IGraph::GetImageInfo

 

Retrieve information about image. The returned information include: width, height, bit depth and format identification of image.

 

bool GetImageInfo(

            const char *name,

            dword &sx,

            dword &sy,

            byte &bpp,

            dword &id

) const;

Parametrs:

name

Pointer to name of image, about which information should be retrieved.

sx, sy

Variables which will contain width and height of image, if the call succeeds, in pixels.

bpp

Variable which will contain bit-depth of image, if the call succeeds, in bytes per pixel.

id

Variable, which will contain unique identifier of image format, if the call succeeds.

Return value:

If the function succeeds, the returned value is true.
If file was not found, or is in unknown format, the returned value is false.

Remarks:

Application may only request information about formats supported by IGraph interface.
