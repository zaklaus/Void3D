IGraph::ProcessWinMessages

Process Window's messages in the main window's message queue. The method internally calls Win32 functions PeekMessage, TranslateMessage and DispatchMessage.

void ProcessWinMessages(

            dword flags = 0

);

Parametres:

flags

Reserved, set this parameter to 0.

Remarks:

This method is supposed to be called in each frame, or sufficiently enough to update Windows processing. No other Windows functions for messages are needed to be called. Additionally this method may dispatch messages directed to modeless dialog boxes, if their handles are added by AddDlgHWND.
