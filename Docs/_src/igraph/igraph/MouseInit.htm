IGraph::MouseInit

Initialize mouse. This method sets mouse to exclusive mode.

bool MouseInit(

            dword mode = 0

);

 

Parametrs:

 

mode

 

Currently unused, set to 0.

 

Return value:

 

The method returns true if the mouse was initialized successfully.

Remarks:

When the main window loses focus, the mouse is automatically unacquired. After receiving focus back, the keyboard is re-acquired. This behavior is transparent to application.
