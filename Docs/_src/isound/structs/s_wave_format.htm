S_wave_format

S_wave_format structure holds information about particular wave format. The structure is defined as follows:

struct S_wave_format{
   dword num_channels;
   dword bytes_per_sample;
   dword samples_per_sec;
   dword size;
};

typedef ISND_INIT *PISND_INIT;

Members:

num_channels

Number of channels in the file. This is 1 for mono, 2 for stereo and 4 for quadro.

bytes_per_sample

Bytes making up single sample.

samples_per_sec

Sound playback frequency, in Hz.

size

Size of sound’s data, in bytes. This member is used in circumstances where it makes sense – for description of particular sound data.
