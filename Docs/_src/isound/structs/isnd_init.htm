ISND_INIT

ISND_INIT structure is used when creating the ISND_driver interface. The structure is defined as follows:

struct ISND_INIT{
   int mix_frequency;
   int num_channels;
   int bits_per_second;
   dword flags;
   void *hwnd;
   void *guid;
   typedef void __stdcall t_log_func(const char*);
   t_log_func *log_func;
};

typedef ISND_INIT *PISND_INIT;

Members:

mix_frequency

Frequency of sound playback, in Hz. Common frequencies of sound devices are 11025, 22050 or 44100 Hz.

num_channels

Number of playback channels. This is 1 for mono or 2 for stereo playback.

bits_per_second

Bits per second of sound playback. Possible numbers are is 8 or 16.

flags

Initialization flags. You may use any combination of following flags:

ISC_3DENABLE

Enable 3D capabilities of underlying sound system.

ISC_VERBOSE

Dump capabilities of sound device to provided callback function log_func.

ISC_USE_EAX

Use EAX capabilities of underlying sound system, if available.

hwnd

Handle of window (HWND), to which the sound playback is applied to. Only when this window is active, the sound output of the driver is audible.

guid

Win32 GUID of device which should be used for sound playback. To use default playback device, you may pass NULL in this parameter.

log_func

Callback function for receiving log messages from ISND_driver.
