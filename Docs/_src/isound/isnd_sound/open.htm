ISND_sound::Open

 

This method loads sound with specified data contained in ISND_source interface.

 

ISND_RESULT Open(

            PISND_source source,

            dword flags = 0

);

Parameters:

source

Pointer to ISND_source interface, which will be used as source for sound playback.

flags

Combination of ISound flags, which specify how will be the sound opened.

Return value:

If successful, the returned value is ISND_OK.
If sound source doesn’t contain any data, the returned value is ISNDERR_NOTINITIALIZED.
