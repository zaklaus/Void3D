ISoundCreate

This exported function is used for creation of ISND_driver interface.

ISND_driver *ISoundCreate(

            ISND_INIT *init_data

);

Parameters:

init_data

Pointer to ISND_INIT structure, which contains parameters for initialization of created ISND_driver interface.

Return value:

If successful, the function returns pointer to newly created and initialized ISND_driver interface.
If the call fails, the returned value is NULL.

See also:

ISND_driver, ISND_INIT
