ISound library

ISound library is responsible for complete sound management – loading of sounds and their conversion into appropriate format, and actual playback through sound hardware.
Master interface is ISND_driver, which creates other interfaces, as well as initializes and shuts down sound hardware, and manages all common tasks. This interface is created by calling IsoundCreate function.

Library: ISound2.dll, ISound2d.dll
Header file: ISound2.h

Interfaces:

ISND_driver

ISND_source

ISND_sound

ISND_3Dsound
