ISND_driver::SetListenerVelocity

 

This method is used for setting logical velocity of sound listener for 3D sound hardware.

 

ISND_RESULT SetListenerVelocity(

            const S_vector &vel

)

Parameters:

vel

New velocity of 3D sound listener, in world coordinates.

Return value:

ISND_OK if the interface was created successfully. For list of error codes see ISound Return Values.

Remarks

The velocity doesn’t change position of sound listener. It is only used for applying special sound effects on 3D sound buffers.
