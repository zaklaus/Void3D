ISND_driver::GetCaps

 

This method is used for retrieving important capabilities about underlying sound system, on which ISound driver operates.

 

ISND_RESULT GetCaps(

            dword *caps

)

Parameters:

caps

Pointer to dword buffer, where capabilities will be stored when function returns. This buffer must have size of ISNDCAPS_LAST elements, and all elements contain values when function call returns. Indices into particular elements of caps are based on E_ISNDCAPS enumerated type.

Return value:

If successful, the returned value is ISND_OK .
If capabilities of underlying sound system cannot be retrieved, the returned is ISNDERR_UNSUPPORTED.
