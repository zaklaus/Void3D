ISound return codes

Following are return codes from ISound library functions.

ISND_OK

This code is returned when the method call was successful.

ISNDERR_NOFILE

Failed to open file.

ISNDERR_OUTOFMEM

Out of memory. The method failed because of low-memory conditions.

ISNDERR_GENERIC

Internal error. This may be caused by unexpected behavior of underlying sound system.

ISNDERR_NOTINITIALIZED

Interface is not properly initialized.

ISNDERR_INVALIDPARAM

Invalid parameters to function call.

ISNDERR_UNSUPPORTED

The method is unsupported.
