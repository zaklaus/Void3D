E_ISNDCAPS

The enumerated type is defined as follows:

enum E_ISNDCAPS{
   ISNDCAPS_NUM_HW_ALL_BUFFERS,
   ISNDCAPS_FREE_HW_ALL_BUFFERS,
   ISNDCAPS_NUM_3D_ALL_BUFFERS,
   ISNDCAPS_FREE_3D_ALL_BUFFERS,
   ISNDCAPS_HW_MEM_ALL,
   ISNDCAPS_HW_MEM_FREE,
   ISNDCAPS_LAST
};

Remarks

 

The members of this enumerated type are indices into array of dword values, which are filled by ISND_driver::GetCaps method. The meaning of data on particular index is following:

ISNDCAPS_NUM_HW_ALL_BUFFERS

Number of available hardware buffers.

ISNDCAPS_FREE_HW_ALL_BUFFERS

Number of free hardware buffers.

ISNDCAPS_NUM_3D_ALL_BUFFERS

Number of available hardware 3D buffers.

ISNDCAPS_FREE_3D_ALL_BUFFERS

Number of free hardware 3D buffers.

ISNDCAPS_HW_MEM_ALL

Total on-card memory, in bytes.

ISNDCAPS_HW_MEM_FREE

Free card memory, in bytes.
