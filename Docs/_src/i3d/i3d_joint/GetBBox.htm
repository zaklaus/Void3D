I3D_joint::GetBBox

Get pointer to bounding-box specifying affect region of the joint.
The box is in its own local coordinates. To transform the box to local coordinates of the joint, use matrix returned by GetBBoxMatrix.

const I3D_bbox *GetBBox() const;

Return Value:

The returned value is a pointer to I3D_bbox structure. The method returns NULL if bounding box is not set.
