I3D_note_callback

I3D_note_callback structure is used for specifying details about note key processed in animated note track. It is declared as follows:

struct I3D_note_callback{
   int curr_time;
   int note_time;
   int stage;
   const char *note;
};

Members:

curr_time

Time, where time animation position of the track currently is set to, in milliseconds.

note_time

Time of note key.

stage

Stage number where the note track is animated.

note

Pointer to string containing the note.
