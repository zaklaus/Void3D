I3DINIT

I3DINIT structure is used when creating the I3D_driver interface. The structure is defined as follows:

struct I3DINIT{
   class IGraph *igraph;
   dword flags;
   class ISND_driver *isound;
   typedef void __stdcall t_log_func(const char*);
   t_log_func *log_func;
};

typedef I3DINIT *PI3DINIT;
typedef const I3DINIT *CPI3DINIT;

Members:

igraph

Pointer to IGraph interface, on which created I3D interface will operate.

flags

Initialization flags. You may use any combination of following flags:

I3DINIT_VERBOSE

During initialization, dump information about capabilities of underlying graphics driver, using callback function log_func.

I3DINIT_DIRECT_TRANS

Do not cache transformed and lit vertices, perform direct TnL.

I3DINIT_WBUFFER

Use w-buffer instead of z-buffer.

I3DINIT_WARN_OUT

Display warnings about unreleased interfaces when releasing the driver interface.

I3DINIT_NO_PSHADER

Disable usage of pixel shaders (simulate that graphics adapter has no pixel shader support).

isound

Pointer to ISND_driver interface, on which created I3D interface will operate.

log_func

Callback function for receiving log messages from I3D_driver.
