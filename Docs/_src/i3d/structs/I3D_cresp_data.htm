I3D_cresp_data

I3D_cresp_data structure is used for provide collision information in collision callback and for modify some of its values for next processing. Just note that most of methods are shared with I3D_collision_data.
The structure is declared as follows:

struct I3D_cresp_data{
   S_vector from;
   S_vector dir;
   void *cresp_context;
   PI3D_volume vol_source;
   const I3D_texel_collision *texel_data;
   S_vector face_normal;
};

Members:

from

Position, from which collision was tested.

dir

Direction of current test. Note that direction may change many times during single test when I3DCOL_SOLVE_SLIDE is specified.

cresp_context

User defined collision context. Passed from collision function to callback functions (I3D_collision_response).

vol_source

Volume sphere which is source of collision (valid only for I3DCOL_SPHERE_GROUP, otherwise NULL).

 texel_data

Pointer to I3D_texel_collision structure of currently tested texel. Valid only in cresp_texel callback function.

face_normal

Normal of face with which we collide. It can difference from GetHitNormal() usually when sphere collide with face edge, or when hit normal was modified in response.

Methods:

Constructors:

I3D_cresp_data();

Default constructor. This method resets all members of the class to zero.

Retrieving values:

float GetHitDistance() const;

Return distance of collision point. Collision distance is normalized (you need normalize dir to compute correct hit position).

 

S_vector ComputeHitPos() const;

Compute and return collision point. It uses formula from + dir*GetHitDistance() / dir.Magnitude(), just make computation instead user.

 

PI3D_frame GetHitFrm() const;

This method returns frame, which was hit.

 

Const S_vector &GetHitNormal() const;

Return normal of collision.

 

Const S_vector &GetDestination() const;

Current destination, where we are going. Should be same as from+dir unless user modify it. Calling this method is applicable only when responsing I3DCOL_SPHERE or I3DCOL_SPHERE_GROUP collision tests.

 

int GetFaceIndex() const;

Index of collision face into frame’s mesh or (–1) if collision frame has no mesh. It can be used for obtain more information like face, facegroup or material.

Modifying values:

 

void ModifyNormal(const S_vector &new_normal);

Change hit normal for further process. Valid only for I3DCOL_SPHERE and I3DCOL_SPHERE_GROUP.

 

void SetDestination(const S_vector &new_dest);

Change destination for further process. Valid only for I3DCOL_SPHERE and I3DCOL_SPHERE_GROUP.
