I3D_bbox

Bounding boxes are axis-aligned boxes represented by min and max corner points. Bounding boxes are used anywhere where axis-aligned bounding box is required. Their primary use is with geometry to perform fast object culling.

I3D_bbox is a class containing two corner edges. It's state may be valid or invalid. For frames in scene which hold geometry they keep geometry extent, for other types they are invalidated.

The bounding box structure is declared as:

struct I3D_bbox{
   S_vector min, max;
};

Accessing bounding box:

User may access directly min and max members of I3D_bbox, which are of type S_vector.

Methods:

const S_vector &operator[](int index) const;

void Invalidate();

bool IsValid() const;

void Expand(S_vector bbox_full[8]) const;

 

Descriptions:

const S_vector &operator[](

            int index

) const;

 

Returns reference to min or max corner of box. Index 0 returns min, index 1 returns max corner.

void Invalidate();

 

Set bounding box to invalidated state. In this state the box is prepared to be expanded by Minimal or Maximal methods of S_vector.

bool IsValid() const;

 

Determines if box is valid.

void Expand(S_vector bbox_full[8]) const;

 

Expand AA bounding-box, defined by 2 extreme points, into a bounding-box defined by

 8 corner points.    

Remarks:

Bounding box is valid if all members of its min vector are smaller than members of max vector.
