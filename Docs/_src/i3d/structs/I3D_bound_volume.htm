I3D_bound_volume

 

I3D_bound_volume is a class which represents bounding volume in 3D space. It contains two representations of bounding volume – bounding box and bounding sphere, and both representations are valid, ideally smallest space, occupied by object.

struct I3D_bound_volume{
   I3D_bbox bbox;
   I3D_bsphere bsphere;
};

Members:

bbox

Bounding box of the volume.

bspehre

Bounding sphere of the volume.
