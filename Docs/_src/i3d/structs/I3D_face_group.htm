I3D_face_group

I3D_face_group is a simple class, containing index of first face into a face list, number of faces, and a pointer to a material that is associated with the group. It is used with I3D_mesh_base.

The class is defined as:

class I3D_face_group{
   PI3D_material mat;
public:
   dword base_index;
   dword num_faces;
 
   I3D_material *GetMaterial();
   const I3D_material *GetMaterial() const;
   void SetMaterial(PI3D_material);
};

Members:

mat

Material associated with the group, note that this is a protected member, and may be accessed only by appropriate functions.

base_index

Index of first face in face list, representing the face group.

num_faces

Number of faces in the group.

I3D_mesh_base, I3D_material
