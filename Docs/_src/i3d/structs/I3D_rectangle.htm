I3D_rectangle

 

I3D_rectangle structure is used for specifying rectangular areas.  The structure is defined as fllows:

struct I3D_rectangle{
   union{
      struct{
         int l, t, r, b;
      };
      struct{
         int m[4];
      };
   };
   I3D_rectangle();
   I3D_rectangle(int l1, int t1, int r1, int b1);
   int &operator[](int i)
   const int &operator[](int i) const;
};

Members:

l, t, r, b

Left, top, right and bottom points specifying rectange.

m

Array for alternate access to four edges of the rectangle.

Constructors:

I3D_rectangle();

Default contructor.

 

I3D_rectangle(int l1, int t1, int r1, int b1);

Constructor initializing the members of the structure into provided values.

Operators

int &operator[](int i);
const int &operator[](int i) const;

Array access operator, accessing members of rectangle as array.
