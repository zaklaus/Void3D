I3D_edge

 
I3D_edge is a structure representing vertex indices of an edge in a vertex mesh. The structure is declared as follows:
 
 

struct I3D_edge{
   word v[2];

   I3D_edge();

   I3D_edge(word v0, word v1);

   const word &operator[](int i);

   word &operator[](int i);
   void Invalidate();
   bool IsValid() const;
};

Members:

v

 

Two vertex indices representing the edge.

 

Constructors:

I3D_edge();

 

Default constructor

 

I3D_edge(word v0, word v1);

 

Constructor initializing members of the structure to provided values.

Operators:

word &operator[](int i);

const word &operator[](int i) const;

 

Array access operator, accessing indices by index. The index value I may be 0 or 1.

Validity:

void Invalidate();

 

Mark edge invalidated. This is a special purpose state, in which first vertex index is set to value 0xffff.

 

bool IsValid() const;

 

Determine if edge is invalidated.
