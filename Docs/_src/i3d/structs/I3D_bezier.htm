I3D_bezier

I3D_bezier template class is a base class for other classes. It contains additional two control points for specifying bezier spline.
This is a template class, which allows to use it for any type which has defined operators for bezier spline computation.

The class is declared as follows:

template<class T>
struct I3D_bezier{

   T in_tan;

   T out_tan;

};

 

Members:

 

in_tan, out_tan

 

Tangent values, representing control points of bezier spline.
