I3D_driver::SetViewport

Setup viewport. This function has meaning only for rendering primitives by methods of I3D_driver or I3D_scene interfaces. It has no effect on viewport of rendered scene, using the I3D_scene::Render call.

I3D_RESULT SetViewport(

            const I3D_rectangle &viewport

);

Parameters:

viewport

New viewport rectangle. See I3D_rectangle for details.

Remarks:

The viewport set by this method is always reset to viewport of scene which was recently rendered, so user must call this method each time before calling user rendering functions, when different view-port is required than those of recently rendered scene.

Return Value:

The returned value is I3D_OK if the call succeeds.
