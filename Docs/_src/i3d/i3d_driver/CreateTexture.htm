I3D_driver::CreateTexture

 

Create I3D_texture interface. The interface is initialized at time of creation.

 

 

I3D_RESULT CreateTexture(

               const I3D_CREATETEXTURE *init_data,

               PI3D_texture *tp

);

Parameters:

init_data

Pointer to initialization structure. See I3D_CREATETEXTURE for details.

tp

Pointer to pointer which will contain valid pointer to I3D_texture interface if the call succeeds.

Return Value:

If the call succeeds, the returned value is I3D_OK.
Otherwise, the returned value is one of I3D_RESULT enumerated type.
