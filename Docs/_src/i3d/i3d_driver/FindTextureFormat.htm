I3D_driver::FindTextureFormat

 

Find texture format most closely matching specified requirements.

 

 

I3D_RESULT FindTextureFormat(

               S_pixelformat &pf,

               dword flags

) const;

Parameters:

pf

Reference to S_pixelformat structure, which will contain information about found texture format when function succeeds.

flags

Flags specifying requirements on texture format to be found. The flags are only hint on which texture format from available ones should be selected.
A combination of following flags may be specified:

FINDTF_PALETIZED

Request for palletized format.

FINDTF_ALPHA

Request for format with alpha-channel.

FINDTF_ALPHA1

Request for format with 1-bit alpha-channel.

FINDTF_COMPRESSED

Request for compressed texture format.

FINDTF_TURECOLOR

Request for true-color format.

FINDTF_BUMPMAP

Request for bump-map format.

FINDTF_RENDERTARGET

Request for format usable as render target.

Return Value:

If the call succeeds, the returned value is I3D_OK.
Otherwise, the returned value is one of I3D_RESULT enumerated type.
