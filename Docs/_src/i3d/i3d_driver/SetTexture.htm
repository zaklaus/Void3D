I3D_driver::SetTexture

 

Set current texture, which will be used for rendering polygons by user rendering functions. The texture set this way will be valid only for calls to I3D_scene::Draw* type of functions.

 

 

I3D_RESULT SetTexture(

            const I3D_texture *texture

);

Parameters:

texture

Pointer to I3D_texture interface representing texture to be used..

Remarks:

This method, as all user-callable rendering functions, supports only single texture stage.

Return Value:

If the call succeeds, the returned value is I3D_OK.

See also:

I3D_texture, I3D_scene
