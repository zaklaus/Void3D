I3D_driver::EnumVisualTypes

 

Enumerate all types of visuals supported by the library.

 

 

I3D_RESULT  EnumVisualTypes (

            I3D_ENUMVISUALPROC *cb_proc,

            dword context

) const;

Parameters:

cb_proc

Callback function, which will be called for each visual type. The function is declared as follows:

I3DENUMRET I3DAPI I3D_ENUMVISUALPROC(dword visual_type, const char *friendly_name, dword context);

Parameters into the callback are:

visual_type

Type of visual, identified by FOUR-CHARACTER code.

Friendly_name

User-friendly name of visual type.

context

User value, passed into the EnumVisuals function.

context

User value passed to the callback function.

Return Value:

The returned value is I3D_OK if the call succeeds, or I3DERR_CANCELED if user canceled enumeration in the callback function.
