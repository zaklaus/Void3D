I3D_driver::GetSoundEnvData

 

Retrieve data of particular sound environment set.

 

 

dword GetSoundEnvData(

            int set_id,

            I3D_SOUND_ENV_DATA_INDEX index

) const;

Parameters:

set_id

Unique integer value, identifying set of environment properties.

index

Index of environment variable in the set. See I3D_SOUND_ENV_DATA_INDEX for details.

Return Value:

Data of variable. The type of data is dependent on register, which is being read.

See Also:

SetSoundEnvData, I3D_SOUND_ENV_DATA_INDEX
