I3D_driver::ClearDirs

 

Clear all directories of particular type.

 

 

I3D_RESULT ClearDirs(

               I3D_DIRECTORY_TYPE dir_type

);

Parameters:

dir_type

A member of I3D_DIRECTORY_TYPE enumerated type, specifying directory class, where new directory is being removed.

Return Value:

If the call succeeds, the returned value is I3D_OK.
Otherwise, the returned value is one of I3D_RESULT enumerated type.

See Also:

AddDir, RemoveDir, NumDirs, GetDir
