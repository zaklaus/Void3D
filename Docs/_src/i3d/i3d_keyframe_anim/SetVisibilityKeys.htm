I3D_keyframe_anim::SetVisibilityKeys

Set up visibility track. This track animates visibility of I3D_frame interface.

I3D_RESULT SetVisibilityKeys(

            I3D_anim_visibility *keys,

            dword num_keys

);

Parameters:

keys

Pointer to array of keys stored in I3D_anim_visibility structures.

num_keys

Number of keys pointed to by keys parameter.

Return Value:

The returned value is I3D_OK if the call completed successfully.

Remarks:

By calling this method, all data currently stored in the visibility track will be overwritten by new values.

See also:

I3D_anim_visibility, I3D_frame::SetOn
