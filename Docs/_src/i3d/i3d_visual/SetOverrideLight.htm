I3D_visual::SetOverrideLight

Setup light color for special lighting mode. Once set, visual use this color instead of standard vertex lighting. This mode is similar to lightmap object lit by ambient with specified color. Only other lights applied on visual are lights with I3DLIGHTMODE_DYNAMIC_LM mode. To restore back standard mode, call function with NULL pointer.
This method is designed for help to make some special visuals properly lit (like bushes and small rocks in lightmap shadow).

void SetOverrideLight(
            const S_vector *light_color
);

Parameters:

Light_color

Value of override light, which will be used instead standard vertex  lightning. Light color is vector with r, g, b parts in range 0.0 to 2.0.
