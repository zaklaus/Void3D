I3D_visual

Remarks:

I3D_visual interface cannot be directly created.
Interfaces inherited from I3D_visual implement their own methods for storing geometry and rendering.

The visibility flag set for this interface enables or disables rendering of this frame and all visual frames linked to this frame.

Properties:
Visual's properties are per-frame data, which are set and retrieved by common functions, regardless of visual type they are performed on. This allows flexible adding and removing of visual types without introducing new methods for setting/retrieving data for each particular type. Each visual type has fixed number of properties which may be determined by I3D_driver::NumProperties. Property on each index has a type and user-friendly name, which may be retrieved by I3D_driver::GetPropertyName and I3D_driver::GetPropertyType.
