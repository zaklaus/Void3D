I3D_collision_response

This function is called from TestCollision function when collision with face or texel was detected.

bool I3DAPI I3D_collision_response(
            I3D_cresp_data &rd

);

 

Parameters:

rd

Reference to structure I3D_cresp_data, which contain collision information and provide some methods for modify them.

Return value:

If user returns true, current collision is accepted with modified values and test ends.
If user returns false, current collision is ignored and test continues with next one.

Remarks:

If I3DCOL_SOLVE_SLIDE was specified in collision test and user return true, default slide collision response is called after this one, and set destination appropriate to slide along collision normal. New dir is generated and tested again until entire length of segment pass successfully without collision or rest of segment is blocked.
