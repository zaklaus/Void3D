I3D_material::SetTexture

Set the texture which should be used when faces with this material are rendered. If this value is NULL, faces are rendered without texture.

void SetTexture(
            I3D_MATERIAL_TEXTURE_INDEX mti,

            PI3D_texture texture

);

Parameters:

mti

A member of I3D_MATERIAL_TEXTURE_INDEX type, identifying texture to set.

texture

Texture which  will be used to render faces using this material.

See Also:

GetTexture, I3D_texture, I3D_MATERIAL_TEXTURE_INDEX
