I3D_SOUND_ENV_DATA_INDEX

This enumerated type represents indices into sound environment variables of single sound environment block. Names of variables, their meaning and type of data associated with variables are as follows:

Variable

Type

Description

Range

I3D_SENV_F_ROOM_LEVEL_LF

float

room level low frequency ratio

0.0 ... 1.0

I3D_SENV_F_ROOM_LEVEL_HF

float

room level high frequency ratio

0.0 ... 1.0

I3D_SENV_I_REVERB_DECAY_TIME

int

reverbation decay time, in msec

 

I3D_SENV_F_DECAY_HF_RATIO

float

HF to LF decay time ratio

0.0 ... 1.0

I3D_SENV_F_REFLECTIONS_RATIO

float

early reflections level - relative to room effect ratio

0.0 ... 1.0

I3D_SENV_I_REFLECTIONS_DELAY

int

early reflections delay, in msec

 

I3D_SENV_F_REVERB_RATIO

float

late reverberation level - relative to room effect

0.0 ... 1.0

I3D_SENV_I_REVERB_DELAY

int

late reverberation delay time relative to initial reflection, in msec

 

I3D_SENV_F_ENV_DIFFUSION

float

environment diffusion ratio

0.0 ... 1.0

I3D_SENV_S_NAME

char*

friendly name of environment

 

I3D_SENV_F_EMUL_VOLUME

float

sector's volume in emulation mode

0.0 … 1.0
