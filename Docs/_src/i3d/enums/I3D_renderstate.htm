I3D_RENDERSTATE

The I3D_RENDERSTATE enumerated type defines indices into internal registers of I3D_driver interface. Following table describes meaning of each register, and type of data.

Register

Type

Description

Quality:

RS_LINEARFILTER

bool

Linear filtering on textures. When enabled, the system chooses either linear filtering or anisotropic filtering, depending on hardware support.

RS_MIPMAP

bool

Mipmapping on textures. This register enables/disables hardware support of mipmapping.

RS_USE_EMBM

bool

Enable/disable environment-mapped bump-mapping.

RS_DETAILMAPPING

bool

Enable/disable detail-mapping.

RS_USEZB

bool

Applying of z-buffer during scene rendering.

RS_CLEAR

bool

Screen clearing before scene rendering.

RS_DITHER

bool

Dithering during polygon rasterization.

RS_WIREFRAME

bool

Force wire-frame rendering.

RS_USESHADOWS

bool

Render shadows.

RS_FOG

bool

Use fog.

RS_LOD_SCALE

float

Quality of level-of-detail selection, multiplier, default is 1.0.

RS_USELMAPPING

bool

Use light-mapping.

RS_LMTRUECOLOR

bool

Choose true-color pixel format for light-map textures, if available. Otherwise high-color (16-bit) pixel format is selected.

RS_LMDITHER

bool

Use dithering if light-map textures are in high-color mode.

RS_ANISO_FILTERING

bool

Use anisotropic filtering.

RS_LM_AA_RATIO

int

Antialias sub-pixel ratio used at light-map creation. This value specifies in how many sub-divisions each lumel in light-map is computed. The more sub-divisions, the better antialiasing on computed lightmap is achieved, but the slower the computation is, due to additional collision testing.
Valid values are in range 1 to 100.

RS_SOUND_VOLUME

float

Set global sound volume applied to all I3D_sound frames. The default value is 1.0f.
The volume of all played sounds is multiplied by this value.

Debugging:

RS_DRAWBOUNDBOX

bool

Debugging - render object bounding boxes as white lines.

RS_DRAWHRBOUNDBOX

bool

Debugging - render hierarchy bounding boxes as green lines.

RS_DRAWLINKS

bool

Debugging - render hierarchy links among objects as orange lines.

RS_DRAWMIRRORS

bool

Render mirrors.

RS_DEBUGDRAWSHADOWS

bool

Visualize shadows.

RS_DRAWSECTORS

bool

Debugging - render sector skeletons as gray polygons.

RS_DRAWPORTALS

bool

Debugging - render sector portals as blue/purple polygons outlined by blue lines.

RS_DRAWVISUALS

bool

Draw visuals.

RS_DEBUGDRAWSHDRECS

bool

Debugging – render shadow receivers.

RS_DRAWVOLUMES

bool

Draw collision volumes.

RS_DRAWLIGHTS

bool

Draw lights as icons.

RS_DRAW_COL_TESTS

bool

Visualize all collision tests.

RS_DEBUGMATVISUALIZE

int

Visualize specified material (set to –1 to disable visualization).

RS_DRAWSOUNDS

bool

Draw sounds as icons.

RS_DRAWLMTEXTURES

bool

Draw textures that are used by engine for light-map cache.

RS_DRAWCAMERAS

bool

Draw cameras as icons.

RS_DRAWDUMMYS

bool

Draw dummies as icons.

RS_DRAWOCCLUDERS

bool

Draw outline of occluders.

RS_DRAWTEXTURES

bool

Use texturing during rendering.

RS_DEBUGDRAWBSP

bool

Visualize BSP collision tree.

RS_DRAWJOINTS

bool

Draw joints as icons and connections to parent frame.

RS_DEBUGDRAWDYNAMIC

bool

Draw dynamic volume tree hierarchy.

RS_DEBUGDRAWSTATIC

bool

Draw static frames.

RS_LOD_INDEX

int

Explicit index of level-of-detail to be rendered. If this register is -1 (default), LOD is computed by current distance and performance criteria. Any other value forces specific LOD to be used.

RS_USE_OCCLUSION

bool

Use occlusion testing.

RS_ENVMAPPING

bool

Use environment mapping.

RS_DEBUG_SHOW_OVERDRAW

bool

Render scene using special technique, in order to show pixel owerdraw.

RS_DEBUG_INT0 … RS_DEBUG_INT7

int

Set up internal integer register used for debugging.

RS_DEBUG_FLOAT0 … RS_DEBUG_FLOAT7

float

Set up internal float register used for debugging.

Loading:

RS_LOADMIPMAP

bool

Build mipmap levels during texture creation (only when requested during texture creation).

RS_TEXTUREDITHER

bool

Dither bitmaps during texture creation.

RS_TEXTURECOMPRESS

bool

Compress textures during loading.
