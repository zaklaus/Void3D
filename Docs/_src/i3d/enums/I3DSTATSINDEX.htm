I3DSTATSINDEX

The I3DSTATSINDEX enumerated type is used to identify, which statistics type is to be retrieved. This table shows possible values, as well as structures associated with particular type, into which statistics will be written.

Some of the indices are applicable in call to I3D_driver::GetStats, while others are applicable in call to I3D_scene::GetStats. You’ll learn particular use under the Description tab.

I3DSTATSINDEX

Description

Associated structure

I3DSTATS_VOLUME

Collision testing
Applicable to: I3D_scene

struct I3D_stats_volume{
   dword sphere_sphere, sphere_rectangle;
   dword sphere_box, sphere_face;
   dword line_sphere, line_rectangle;
   dword line_box, line_face;
};

Members:

sphere_sphere

Number of sphere-to-sphere collision tests.

sphere_rectangle

Number of sphere-to-rectangle collision tests.

sphere_box

Number of sphere-to-box collision tests.

sphere_face

Number of sphere-to-polygon collision tests.

line_sphere

Number of line-to-sphere collision tests.

line_rectangle

Number of line-to-rectangle collision tests.

line_box

Number of line-to-box collision tests.

line_face

Number of sphere-to-polygon collision tests.

Remarks:

After retrieving these stats, the internal counters are reset to zero.

Rendering

struct I3D_stats_render{

   dword line;
   dword vert_trans;
};

Members:

Number of triangles rendered.

line

Number of lines rendered.

vert_trans

Number of vertices transformed/sent to graphics card.

Remarks:

After retrieving these stats, the internal counters are reset to zero.

I3DSTATS_SCENE

Scene
Applicable to: I3D_scene

struct I3D_stats_scene{
   dword frame_count[FRAME_LAST];
   dword num_vis_computed;
   dword dyn_shd_casters;
};

Members:

frame_count

Number of visuals of particular type rendered.

num_vis_computed

Number of visuals preprocessed.

dyn_shd_casters

Number of dynamic shadows rendered.

Remarks:

After retrieving these stats, the internal counters are reset to zero.

I3DSTATS_VIDEO

Video memory usage
Applicable to: I3D_driver

struct I3D_stats_video {
   dword txt_total;
   dword txt_vidmem;
   dword txt_sys_mem;
   dword txt_video_mem;
   dword vidmem_total;
   dword vidmem_free;
   dword txtmem_total;
   dword txtmem_free;
};

Members:

txt_total

Number of textures currently created.

txt_vidmem

Number of textures loaded into video memory.

txt_sys_mem

System memory used by textures.

txt_video_mem

Video memory used by textures.

vidmem_total

Total bytes of video memory.

vidmem_free

Free bytes of video memory.

txtmem_total

Total bytes of texture video memory.

txtmem_free

Free bytes of texture video memory.

I3DSTATS_MEMORY

Memory usage
Applicable to: I3D_driver

struct I3D_stats_memory{
   dword available;
   dword used;
   dword used_other;
   dword num_blocks;
   dword dbase_size;
   dword dbase_used;
};

Members:

available

Memory available (by system).

used

Memory used by I3D library.

used_other

Memory used by other resources.

num_blocks

Number of allocated memory blocks.

dbase_size

Total bytes available in database.

dbase_used

Bytes used in database.

I3DSTATS_SOUND

Sounds
Applicable to: I3D_scene

struct I3D_stats_sound{
   dword buf_hw_all, buf_hw_free;
   dword buf_3d_all, buf_3d_free;
   dword hw_mem_all, hw_mem_free;
   dword snds_all, snds_audible;
   dword snds_in_card;
   dword mem_used;
};

Members:

buf_hw_all

Number of all hardware sound buffers.

buf_hw_free

Number of free hardware sound buffers.

buf_3d_all

Number of all 3D hardware sound buffers.

buf_3d_free

Number of free 3D hardware sound buffers.

hw_mem_all

Number to total hw memory bytes.

hw_mem_free

Number to free hw memory bytes.

snds_all

Total number of sounds currently created.

snds_audible

Total number of sounds currently audible.

snds_in_card

Number of sounds uploaded to sound card.

mem_used

System memory used by all sounds.
