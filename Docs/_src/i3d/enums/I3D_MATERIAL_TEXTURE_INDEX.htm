I3D_MATERIAL_TEXTURE_INDEX

The I3D_MATERIAL_TEXTURE_INDEX enumerated type specifies texture assigned to material. Following values are defined:

MTI_DIFFUSE

Texture used for base texture.

MTI_ENVIRONMENT

Texture used for environment mapping.

MTI_EMBM

Texture used for environment-mapped bump-mapping.

MTI_DETAIL

Texture used for detail-mapping.

MTI_NORMAL

Texture used as normal-map for bump-mapping effects.

MTI_SECONDARY

Texture used as secondary diffuse color, blended with base texture using a blend map.
