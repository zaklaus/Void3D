I3D_DIRECTORY_TYPE

The enumerated type I3D_DIRECTORY_TYPE is defined as follows:

enum I3D_DIRECTORY_TYPE{

   I3DDIR_MAPS,
   I3DDIR_SOUNDS,
   I3DDIR_PROCEDURALS,
};

 

I3DDIR_MAPS

 

This type describes directory for storing bitmaps.

 

I3DDIR_SOUNDS

 

This type describes directory for storing sounds.

 

I3DDIR_PROCEDURALS

 

This type describes directory for storing procedurals.
