I3D_ANIMATION_TYPE

The I3D_ANIMATION_TYPE is used for specifying type of animation derived class. It is defined as follows:

enum I3D_ANIMATION_TYPE{
   I3DANIM_KEYFRAME,
   I3DANIM_POSE,
};

I3DANIM_KEYFRAME

This type describes animation consisting of key-frame tracks.

I3DANIM_POSE

This type describes animation made of fixed pose.
