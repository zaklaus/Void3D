Visual types

This page describes types of visual objects and special properties associated with them. The properties are assigned using I3D_visual::SetProperty, and may be read using I3D_visual::GetProperty functions.

Particle

Four-CC code: I3D_VISUAL_PARTICLE = PRTC

Property name

Value type

Desciption

I3DPROP_PRTC_I_OUT_TIME

int

Base and random time after which new elements are created.

I3DPROP_PRTC_I_OUTT_THRESH

int

I3DPROP_PRTC_I_LIFE_LEN

int

Base and random time – length of element’s life.

I3DPROP_PRTC_I_LIFEL_THRESH

int

I3DPROP_PRTC_F_SCALE_GROW_MIN

float

Min and max range specifyinghow elements may be scaled.

I3DPROP_PRTC_F_SCALE_GROW_MAX

float

I3DPROP_PRTC_F_INIT_POS_THRESH

float

Thresh of new element’s initial position.

I3DPROP_PRTC_F_INIT_DIR_THRESH

float

Thresh of new element’s initial direction.

I3DPROP_PRTC_I_AFFECT_TIME

int

Base and random time of period how often direction is changed.

I3DPROP_PRTC_I_AFFECTT_THRESH

int

I3DPROP_PRTC_F_AFFECT_DIR_THRESH

float

Random factor of direction change.

I3DPROP_PRTC_F_OPACITY_IN

float

Ratio of life (0.0 … 1.0), until which opacity fades in and out during life-time of element.

I3DPROP_PRTC_F_OPACITY_OUT

float

I3DPROP_PRTC_F_BASE_DIR_X

float

Base direction.

I3DPROP_PRTC_F_BASE_DIR_Y

float

I3DPROP_PRTC_F_BASE_DIR_Z

float

I3DPROP_PRTC_F_DEST_DIR_X

float

Destination direction.

I3DPROP_PRTC_F_DEST_DIR_Y

float

I3DPROP_PRTC_F_DEST_DIR_Z

float

I3DPROP_PRTC_E_DRAW_MODE

int

Render mode: 0=opaque, 1=alphablend, 2=add.

I3DPROP_PRTC_I_SETCOUNT

int

Number of elements to emit, set zero for unlimited.

I3DPROP_PRTC_B_BASE_DIR_WORLD

bool

Base direction is in world coordinates, rather than local.

I3DPROP_PRTC_B_DEST_DIR_WORLD

bool

Destination direction is in world coordinates, rather than local.

I3DPROP_PRTC_B_EMIT_BASE

bool

If set to true, particle emits initial set of elements when it is created.

I3DPROP_PRTC_F_ROT_SPEED_MIN

float

Rotation speed range, in radians/sec. Negative values rotate to left, positive to right.

I3DPROP_PRTC_F_ROT_SPEED_MAX

float

I3DPROP_PRTC_F_ELEMENT_SCALE

float

Scale of element. By default, size of element’s rectangle’s side is 1.0, this is multiplier.

I3DPROP_PRTC_F_OPACITY

float

Opacity, value from 0.0 to 1.0.

I3DPROP_PRTC_E_MODE

int

Alignment mode: 0=billboard, 1=XY, 2=XZ, 3=YZ.

I3DPROP_PRTC_B_OPTIMIZED

bool

Optimized mode, not animating if out of view.

I3DPROP_PRTC_MATERIAL

PI3D_material

Material used for rendering particle.

I3DPROP_PRTC_I_NUMELEMENTS

int

Number of elements currently created (read-only).

 

Billboard

Four-CC code: I3D_VISUAL_BILLBOARD = BBRD

Property name

Value type

Desciption

I3DPROP_BBRD_B_AXIS_MODE

bool

Use axis-mode particle.

I3DPROP_BBRD_I_ROT_AXIS

int

Rotation axis for axis mode (0=X, 1=Y, 2=Z).

 

Flare

Four-CC code: I3D_VISUAL_FLARE = FLRE

Property name

Value type

Desciption

I3DPROP_FLRE_F_SCALE_DIST

float

Distance to camera, from which flare starts to scale down.

I3DPROP_FLRE_F_ROTATE_RATIO

float

Ratio how many times is flare rotated as distance to camera changes, in turns/meter

I3DPROP_FLRE_F_NORMAL_ANGLE

float

Angle from local Z axis at which flare is fade out. Set to 0 to disable this fading.

I3DPROP_FLRE_I_FADE_TIME

int

Time how long flare fades in/out when occluded. Set to 0 to disable and use Z-buffered rendering.

I3DPROP_FLRE_F_PIVOT

float

Pivot displacement used for preventing flare to cut to close objects.

Property name

Value type

Desciption

I3DPROP_LMAP_F_RESOLUTION

float

Resolution of light-map, in texels per unit geometry distance.

Property name

Value type

Desciption

float

Shift values for U and V coordinates. The meaning of these values is dependent on mode.

I3DPROP_OUVS_F_SHIFT_V

float
