I3D_LOAD_CB_PROC

This prototype specifies callback function used during opening data through certain I3D interfaces. It is declared as:

bool I3DAPI I3D_LOAD_CB_PROC(

            I3D_LOADMESSAGE msg,

            dword prm1,

            dword prm2,

            void *context

);

Parameters:

msg

Callback message. This may be one of following:

CBM_PROGRESS

Opening progress. This message is sent only if user specified I3DLOAD_PROGRESS flag in the opening method.
Parameters are used as follows:

prm1 … float progress (bit-mask of float value, ranging from 0.0 to 1.0)
prm2 … loading phase (0=beginning, 1=progress, 2=end)

If the returned value for this message is true, the loading is canceled, and the loader method returns I3DERR_CANCELED.

CBM_ERROR

An error message encountered during loading is reported. This message is sent only if user specified I3DLOAD_ERRORS flag in the opening method.
Parameters are used as follows:

prm1 … const char *message – error message being reported
prm2 … int importance (0=error, 1=warning)

CBM_LOG

Loading log. This message is sent only if user specified I3DLOAD_LOG flag in the opening method.

Parameters are used as follows:

prm1 … const char *message – log message being reported
prm2 … unused

prm1, prm2

Additional parameters, whose usage depends on message being sent.

context

User context value passed to the opening function.

Return value:

The return value depends on message being sent. When message doesn’t specify particular meaning of return value, return false.
