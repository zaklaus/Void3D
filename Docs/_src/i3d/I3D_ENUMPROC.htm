I3D_ENUMPROC

I3D_ENUMPROC is a prototype of I3D callback function, used for enumeration of frames.

I3DENUMRET I3DAPI I3D_ENUMPROC(

            PI3D_frame frm,

            dword context

);

Parameters:

frm

Pointer to I3D_frame interface being enumerated. 

context

User context value passed to EnumFrames.

Return value:

See I3DENUMRET enumerated type for possible return values from the callback function.

See also:

I3DENUMRET, I3D_frame
