I3D_model::SetAnimCurrTime

Set current animation time of particular stage.

I3D_RESULT SetAnimCurrTime(

            dword stage,

            dword time

);

Parameters:

stage

Animation stage, which is being updated.

time

New interpolation time on the stage.

Return value:

The returned value is I3D_OK if the call succeeded.
