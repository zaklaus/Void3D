I3D_model::SetAnimation

Set animation for model. See I3D_animation_set for detail on how animation is linked to model.
The last 3 parameters specify operations performed in animation pipeline.

I3D_RESULT SetAnimation(

               dword stage,

               const I3D_animation_set *animation,

               dword options = I3DANIMOP_SET,

               float scale_factor = 1.0f,

               float blend_factor = 1.0f,

               float speed = 1.0f

);

Parameters:

stage

Animation stage to use for this animation.

animation

Animation being set.

options

Options specifying how this animation will be blended with other stages.

Value applying mode:

                I3DANIMOP_SET = replace previous value by new value

    3DANIMOP_ADD = add new value (scaled by scale factor) with previous value

   3DANIMOP_BLEND = blend new value (scaled by scale factor) with previous value using blend factor

   I3DANIMOP_DAMP_ADD = damp previous value by damp factor, and add current value

   I3DANIMOP_APPLY_MASK = bitmask of the applying mode bits

   3DANIMOP_BRACE_MASK = bitmask of brace count

   I3DANIMOP_BRACE_SHIFT

                  number of bits to shift precedence braces count

   I3DANIMOP_LOOP

                  loop animation (if not specified, anim is played only once and stopped)

   I3DANIMOP_KEEPDONE

                  do not erase finished animations

   I3DANIMOP_FINISH_NOTIFY

                  notification through callback requested when animation in the stage finished playback

scale_factor

Factor how results of this animation will be scaled.

blend_factor

Factor how this animation will be blended with other stages.

speed

Speed multiplier for playback of this animation.

Return Value:

The returned value is I3D_OK if the call succeeded.
