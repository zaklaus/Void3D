I3D_frame::SetFlags

 

Set special-purpose flags associated with frame. Using this method, you may select which bits of the special-purpose double-word you want to update, and value of those new bits.

 

dword SetFlags(

            dword new_flags,

            dword flags_mask = 0xffffffff

);

Parameters:

new_flags

New bit values to set.

flags_mask

Bit mask, in which bits will be changed.

Return Value:

The returned value is new value of special-purpose dword register.

Remarks:

Following bit values are currently defined:

Description
