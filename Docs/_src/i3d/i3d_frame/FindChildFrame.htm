I3D_frame::FindChildFrame

 
Find child of frame matching specified criteria

 

PI3D_frame FindChildFrame(

            const char *name,

            dword flags = ENUMF_ALL

) const;

Parameters:

name

Name of child to find.

data

Flags specifying how the child is searched.

Return Value:

The returned value is pointer to found frame, or NULL if no frame was found.

Remarks:

See EnumFrames for details on how name and flags parameters are used.
