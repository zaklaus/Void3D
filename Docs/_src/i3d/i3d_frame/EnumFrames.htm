I3D_frame::EnumFrames

 
Enumerate all children of frame matching specified criteria.

 

I3D_RESULT EnumFrames(

            I3D_ENUMPROC *ep,

            dword context,

            dword flags = ENUMF_ALL,

            const char *mask = NULL

) const;

Parameters:

ep

The callback function.

context

User context value, passed to callback function.

flags

Flags specifying enumeration criteria. Following flags are defined:

ENUMF_VISUAL - Enumerate frames of type FRAME_VISUAL
ENUMF_LIGHT - Enumerate frames of type FRAME_LIGHT
ENUMF_CAMERA - Enumerate frames of type FRAME_CAMERA
ENUMF_SOUND - Enumerate frames of type FRAME_SOUND
ENUMF_SECTOR - Enumerate frames of type FRAME_SECTOR
ENUMF_DUMMY - Enumerate frames of type FRAME_DUMMY
ENUMF_MODEL - Enumerate frames of type FRAME_MODEL
ENUMF_VOLUME - Enumerate frames of type FRAME_VOLUME.
ENUMF_USER - Enumerate frames of type FRAME_USER.
ENUMF_VOLUME - Enumerate frames of type FRAME_VOLUME.
ENUMF_JOINT - Enumerate frames of type FRAME_JOINT.
ENUMF_OCCLUDER - Enumerate frames of type FRAME_OCCLUDER.
ENUMF_ALL - Combination of all previous flags.
ENUMF_WILDMASK - Use wild-mask name search.

mask

Mask, specifying the name of frames to enumerate. If this is used with ENUMF_WILDMASK flag, any ? and * characters in the mask string will be used for wild-char comparision.

Return Value:

The returned value is I3D_OK if the enumeration completed successfully, or I3DERR_CANCELED if user canceled enumeration.

Remarks:

Each frame may have other frames attached to itself in hierarchy tree. The user may enumerate the attached objects in the tree using this method. Provided call-back functions is called for each child frame matching specified criteria.
