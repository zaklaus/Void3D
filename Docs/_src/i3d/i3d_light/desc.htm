I3D_light

I3D_light represents light object. This interface inherits from I3D_frame interface.

Light and sectors:
Certain light is applied only to sectors on which it is registered. See I3D_Sector::AddLight and I3D_sector::RemoveLight for details.

I3D_light::SetColor

 

Set color of light. This is a vector with red, green and blue values, which are normalized to range 0.0 to 1.0. The values out of this range may be specified too, in case of negative value the light becomes dark light, in case of positive values above 1.0 the light has greater intensity. The default color of light is white.

 

void SetColor(

            const S_vector &color

);

Parameters:

color

Red, green and blue components of color stored in S_vector structure.

See Also:

GetColor

I3D_light::SetLightType

 

Set type of light.

 

I3D_RESULT SetLightType(

            I3D_LIGHTTYPE type

);

Parameters:

type

Following light types of type I3D_LIGHTTYPE are defined:

I3DLIGHT_POINT

Point light which radiate in same ratio to all directions.

I3DLIGHT_POINTAMBIENT

Ambient light, which has position and ranges.

I3DLIGHT_SPOT

Spot light, which has range and a radiation cone.

I3DLIGHT_DIRECTIONAL

Directional light. This light type has no range.

I3DLIGHT_AMBIENT

Ambient light illuminates all geometry by same amount regardless of its position or orientation.

I3DLIGHT_FOG

Fog light.

I3DLIGHT_LAYEREDFOG

Layered fog.

Return Value:

The returned value is I3D_OK if the call succeeded.

See Also:

GetLightType

I3D_light::SetMode

 

Set mode how this light operates. The flags specified in call to this method specify, which kind of objects will this light illuminate.

 

I3D_RESULT SetMode(

            dword mode

);

Parameters:

mode

One or more of following flags may be specified:

I3DLIGHTMODE_VERTEX

Light applies on vertices of mesh-based visuals (except of lightmaps).

I3DLIGHTMODE_LIGHTMAP

Light is used during creation of light-maps.

I3DLIGHTMODE_SHADOW

This light is used for computation of real-time shadows.

I3DLIGHTMODE_DYNAMIC_LM

This light will affect vertices of light-mapped visuals.

Return Value:

The returned value is I3D_OK if the call succeeded.

See Also:

GetMode

I3D_light::SetPower

 

Set power of light. The resulting light of color will be color set by SetColor, and power multiplier set by SetPower. Using this method is a preferred way to define darkness or lightness lights. The default power of light is 1.0.

 

vois SetPower(

            float power

);

Parameters:

power

New power multiplier.

See Also:

GetPower

I3D_light::SetRange

Set near and far range of light. The intensity between light position and near range is at 100%, then it linearly fades out between near and far range and has no intensity beyond far range. The default range of light is 1.0, 100.0.

Fog light: the near is a normalized value in range 0.0 to 1.0 and means ratio between camera position and far fog range, where fog starts. The far value is distance, in meters, from camera, where fog ends. If this value is 0.0, this value is determined directly from far clipping plane of current camera.

 

vois SetRange(

            float near,

            float far

);

Parameters:

near, far

Near and far range of light.

See Also:

GetRange

I3D_light::SetCone

Set the cone of spot-light. The spot-light has cone aligned with its local Z axis, and two applies are used – inner and outer cone angle, which specify intensity applied onto lit objects based on their location within the cone. The intensity inside the inner cone is at 100%, then it linearly fades out between near and far cone angle and has no intensity beyond the outer cone.

vois SetCone(

            float inner,

            float outer

);

Parameters:

inner, outer

Inner and outer cone angles, in radians.

See Also:

GetCone
