I3D_volume::SetVolumeType

 
Setup type of collision volume.
 

I3D_RESULT SetVolumeType(

            I3D_VOLUMETYPE type

);

Parameter:

type

Type of collision volume. This may be one of following values:

I3DVOLUME_SPHERE

Spherical volume. Radius is given solely by frame’s scale (locally it is 1.0).

I3DVOLUME_RECTANGLE

Rectangular volume. Dimensions are given by non-uniform scale x and y members.

I3DVOLUME_BOX

Box volume. Dimensions are given by non-uniform scale vector.

I3DVOLUME_CYLINDER

Cylinder volume. Dimensions are given by non-uniform scale x (radius) and z (half length) members.

I3DVOLUME_CAPCYL

Capped cylinder volume. Dimensions are given by non-uniform scale x (radius) and z (half length without cap) members.

Return Value:

The returned value is I3D_OK if the call was successful.

See Also:

GetVolumeType

I3D_volume::SetCategoryBits
I3D_volume::SetCollideBits

Set category collision category bits.
Category bits are used for early accepting or rejecting of volumes during collision testing, based on matching bits. These bits specify category in which volume falls to. The meaning of bits is user-defined.
Collision bits are used in active collision tests (e.g. contact generation) for testing common category bits with tested volume.

void SetCategoryBits (

            dword bits

) const;

void SetCollideBits (

            dword bits

) const;

Parameter:

bits

New category/collision bits.
