I3D_CALLBACK

I3D_CALLBACK is a prototype of I3D callback function, used by various methods.

void I3DAPI I3D_CALLBACK(

            I3D_CALLBACK_MESSAGE msg,

            dword prm1,

            dword prm2,

            void *context

);

Parameters:

msg

Message, which may be one of following:

I3DCB_MESSAGE, I3DCB_DEBUG_MESSAGE

String messages, used for debugging purposes. Parameter prm1 is pointer to string.

I3DCB_DEBUG_CODE

User-defined debug code, used for debugging purposes.

I3DCB_ANIM_FINISH

Sent to callback passed to I3D_interpolator::Tick when animation in a stage completes. Parameter prm1 is pointer to I3D_frame interface that is being animated, and prm2 is stage number, which has finished playing. Called only for frames with corresponding flag (I3DCB_ANIM_FINISH). See I3D_frame::SetFlags for more details.

I3DCB_ANIM_NOTE

Sent to callback passed to I3D_interpolator::Tick when note track encountered note key. Parameter prm1 is pointer to I3D_frame interface that is being animated, and prm2 is constant pointer to I3D_note_callback structure holding details about the note key. Called only for frames with corresponding flag (I3D_FRMF_ANIM_NOTE). See I3D_frame::SetFlags for more details.

 

prm1, prm2

Parameters, which meaning is dependent on message being set.

cb_context

User context value, which was passed along the callback pointer.
