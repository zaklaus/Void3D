I3D_anim_pose::SetPower

 

Set power of pose.

 

 

void SetPower(

            const float *power

);

Prameters:

power

New power to assign pose to. When this parameter is NULL, the power is set to un-initialized.

Remarks:

The power value represents scale, volume (FRAME_SOUND) or lightness (FRAME_LIGHT), dependent on type of frame being animated.
