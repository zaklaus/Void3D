I3D_container::Open

Open resources from disk. This method creates interfaces by loading data from disk, and stores it into structures of the container. In case of container of  I3D_model, the root of scene hierarchy of loaded scene is model’s frame itself. In case of container of I3D_scene, the root of scene hierarchy is scene’s Primary Sector.
Internally this method calls Close before starting loading the file and creating new items.

I3D_RESULT Open(

            const char* filename,

            dword flags = 0,

            I3D_LOAD_CB_PROC *cb_proc   = NULL,

            void *cb_context = NULL

);

Parameters:

filename

Name of file to be opened.

flags

Flags specifying behavior. Following flags are available:

I3DLOAD_LOG

Dump debug messages using I3DLOAD_LOG message sent to callback function.

I3DLOAD_PROGRESS

Call callback function with CBM_PROGRESS messages to indicate loading progress.

I3DLOAD_ERRORS

Report loading errors using CBM_ERROR message sent to callback function.

cb_proc

Callback function. See I3D_LOAD_CB_PROC for details.

cb_context

Context value passed to the callback function.

Return value:

The returned value is I3D_OK if the call completed successfully.

Remarks:

All frames created by this method will become owned by the container and will be released when I3D_container::Close is called or when this class is released, or when user calls I3D_container::DeleteFrame with specified frame as parameter.

I3D_LOAD_CB_PROC, Close
