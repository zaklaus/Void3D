I3D_container::Tick

Update container's interpolators and frames, which need updating.

I3D_RESULT Tick(

            int time,

            PI3D_CALLBACK cb_proc = NULL,

            void *cb_context = NULL

);

Parameters:

time

Time, by which to update all interpolators, in milliseconds.

cb_proc

Callback function, which will be used for receiving messages from special animation tracks, as well as notification messages when animation in a track completes. For details, see I3D_CALLBACK.

cb_context

Context value passed to the callback function.

Return value:

If there're no active interpolators, the returned value is I3D_DONE, otherwise the return value is I3D_OK.
