I3D_container::RemoveFrame

Remove frame from internal list of frames owned by this container. If specified frame was found in the list, its reference counter is decreased and it will be removed from the list.

I3D_RESULT RemoveFrame(

            PI3D_frame frm

);

Parameters:

frm

Pointer to I3D_frame interface, which will be removed from container’s list.
