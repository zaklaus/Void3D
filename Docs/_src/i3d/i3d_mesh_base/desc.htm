I3D_mesh_base

The I3D_mesh_base interface physically contains vertices and faces describing geometry. It also contains definition of used material(s). Because of one mesh may have multiple materials, there is one or more groups of faces using the same material. I3D_mesh_base keeps list of face groups.

Return Value:

I3D_mesh_base::LockVertices

 

Obtain access to vertices. To get the access, vertices must get locked. After finishing operation with vertices, user must call UnlockVertices.

 

const S_vector *LockVertices();

Return Value:

The returned value is a constant pointer to array of S_vector structures.

Remarks:

The returned pointer points to the position component of the first vertex. To get access to subsequent position components of next vertices, you must use vertex stride for computing proper pointer of the vertex, not using the returned pointer as array. To get vertex stride, use GetSizeOfVertex method.

See Also:

UnlockVertices, GetSizeOfVertex, S_vector

 

I3D_mesh_base::UnlockVertices
