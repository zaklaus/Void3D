Vertex Component Flags

Vertex component flags are used to specify components contained in vertex. Following flags are defined:

Flag

Data type

Description

I3DVC_XYZ

S_vector

3D position.

I3DVC_XYZRHW

S_vectorw

Transformed 3D position, with reciprocal w value in fourth component.

I3DVC_ALPHA

float

Alpha value.

I3DVC_BETA_MASK
I3DVC_BETA_SHIFT

float

Macros for specifying number of beta components. The number of components must be shifted left by I3DVC_BETA_SHIFT bits.

I3DVC_NORMAL

S_vector

Unit-length normal.

I3DVC_DIFFUSE

dword

Diffuse color.

I3DVC_SPECULAR

dword

Specular color.

I3DVC_TEXCOUNT_MASK
I3DVC_TEXCOUNT_SHIFT

I3D_text_coor

Macros for specifying number of texture coordinates. The number of coordinates must be shifted left by I3DVC_TEXCOUNT_SHIFT bits.

Remarks:

I3DVC_XYZ and I3DVC_XYZRHW flags are mutually exclusive. I3DVC_XYZ is used for specifying non-transformed position, while I3DVC_XYZRHW is used for specifying position transformed to screen coordinates.

Vertex components must be in order specified by the above table, for example specular color must appear after diffuse color. Anyway, except of position, any other components are optional.
