Data File Access

All access to data files in Insanity3D system is made through simple set of functions, which perform open, close, read, write and position functions on files. The system-specific access to files is hidden by implementation, so files may be stored on remote path (CD-ROM), cartridge, compressed in one huge data-file, or stored on hard-disk in a directory structure.

The stream is accessible through C_dta_stream interface.
There’re two possibilities to link with the library – either using dynamic-link library (DLL), which provides access to same DTA files for all modules, or to use static-link library, which links directly to current module, and removes the need to link with DLL. The second option is suitable for stand-alone modules consisting of single executable file. User chooses the static-link version by defining DTA_READ_STATIC symbol before including the dta_read.h header file.

Library: Dta_read.dll, dta_read_static.lib
Header file: dta_read.h
