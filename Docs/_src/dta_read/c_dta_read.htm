C_dta_read

This class is a logical link to archive file containing other data files. All the files in the archive are accessible by C_dta_stream class, until this interface is not released and all handles to its files are closed.

The class is defined as follows:

class C_dta_read{
public:
   virtual unsigned long DTAAPI AddRef() = 0;
   virtual unsigned long DTAAPI Release() = 0;
};

Methods:

The class contains only two methods used for reference-counted manipulation with the interface. When the interface is destroyed (its reference counter reaches zero), the files contained in its archive become inaccessible.
