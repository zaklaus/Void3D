C_dta_stream

This class represents a stream used to read from (or write to) file stored in DTA file. This class is designed to work with reference-counting, and performs basic functions associated with accessing the file.

Methods:

int Read(
            void *buffer,
            dword len
);

Read from the file stream.

Parameters:

buffer

Buffer where data will be stored.

len

Number of bytes to read.

Return value:

The returned value is number of bytes actually read, or –1 if error occurred.

 

int Seek(
            int pos,
            int whence
);

Change current read/write position within the buffer.

Parameters:

pos

Position to set within stream, in bytes. This position is relative to offset specified by whence parameter.

whence

Relative offset. This parameter may be one of following:

DTA_SEEK_SET

Position is relative to beginning of file.

DTA_SEEK_CUR

Position is relative to current position.

DTA_SEEK_END

     Position is relative to end of file.

Return value:

The returned value is new read/write position in the stream, from beginning of the file, in bytes.

 

int GetSize() const;

Get current size of file.

Return value:

The returned value the size of file, in bytes.

 

bool GetTime(
            dword time[2]
) const;

Get last time when file was written to.

Parameters:

time

Pointer to array of two dword values, where the time will be saved.

Return value:

If the call succeeds, the time parameter contains OS-specific time value, and the return value is true.
If the call fails, the return value is false.

 

int Write(
            const void *buffer,
            dword len
);

Parameters:

buffer

Pointer to memory buffer, which will be written to file.

len

Number of bytes which be written.

Return value:

The returned value is number of bytes successfully written to the file.

 

C_dta_read *GetDtaPack() const;

Return value:

The returned value is pointer to C_dta_read interface, in whose opened file reside.
If the file is located directly on disk, the returned value is NULL.
