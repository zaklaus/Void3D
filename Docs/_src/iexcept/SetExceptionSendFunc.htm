SetExceptionSendFunc

This method sets callback function, which will be called when user chooses Send button on the exception dialog. This lets application possibility to sent the crash report to appropriate place, for example save it onto programmers’ shared folder, or send it through e-mail.
If application doesn’t specify this callback function, the Send button on the dialog will be disabled.

void SetExceptionSendFunc(

               bool (__stdcall *func)(const char *msg)

);

Parameters:

func

Pointer to function with specified prototype. The type is:

bool __stdcall ExceptionSendFunc(const char *msg);

Where parameter msg is a string with the same information as appears in the exception dialog box.
It is on application to perform actual sending of the string to appropriate place, with any other necessary information.
