InitializeExceptions

This function configures exception handles so that hardware exceptions are caught by the library, overriding default Windows handler, and appropriate dialog box is displayed. This method also sets FPU control word, in order to mask required floating-point exceptions.

void InitializeExceptions(

               dword fpu_cw = 0xffffffff

);

Parameters:

fpu_cw

Value of FPU control word’s interrupt execution bits. See float.h for details. This bits are masked by _MCW_EM bitmask.
A value of zero bit enables exception handling, while value of one disables it. Thus the default value disables all FPU exceptions.
